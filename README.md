## Project Overview
This project develops a back-end product catalog for an e-commerce website with CRUD operations, user engagement features, and cloud deployment capabilities.

## Stakeholder Requirements
- Create, retrieve, update, and delete products
- Like/dislike products
- List and search products
- Cloud hosting with automated deployments
